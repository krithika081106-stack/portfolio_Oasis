# 🌐 Krithika Portfolio Website

## 📌 Project Overview

This is a **personal portfolio website** designed and developed using **HTML and CSS**. The portfolio showcases academic details, technical skills, projects, internship experience, and contact information in a clean and modern UI.

The design follows a **minimal dark theme with glassmorphism effects**, making it visually appealing and user-friendly.

---

## 🎯 Features

* 👩‍💻 **Personal Introduction** – Displays name and career objective
* ✨ **About Section** – Brief description of interests and goals
* 🎓 **Education Details** – Academic qualifications and scores
* 💻 **Skills Section** – Technical skills displayed as stylish tags
* 📁 **Projects Section** – Highlights key project work
* 💼 **Internship Experience** – Professional exposure details
* 📜 **Certifications** – Courses and achievements
* 🌟 **Extracurricular Activities** – Additional involvement
* 📞 **Contact Section** – Email and social links
* 🎨 **Modern UI** – Glass effect cards, hover animations, gradient background

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the webpage
* **CSS3** – Styling, layout, and animations
